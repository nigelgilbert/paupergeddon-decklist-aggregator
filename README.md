# MTG Paupergeddon Italy Decklist Aggregator

Aggregates Magic: The Gathering Paupergeddon Italy 2025 decklists and outputs card frequencies.

## Usage

```bash
node src/aggregate_decklists.js day_1 > day_1.txt
node src/aggregate_decklists.js day_2 > day_2.txt
```

## Output

```
47 Lightning Bolt
42 Counterspell
38 Brainstorm
```

Cards sorted by frequency (most popular first).
